# Markdown Notes Application

A full-stack Markdown Notes App built with React, Node.js, Express, and SQLite.

## Features

- Create notes
- Edit notes
- Delete notes
- Search notes
- Markdown live preview
- Split screen editor
- Auto save
- Responsive design
- Dark mode
- Pinned notes

## Tech Stack

Frontend:

- React.js
- Vite
- Tailwind CSS
- Axios
- React Markdown

Backend:

- Node.js
- Express.js

Database:

- SQLite

---

## Local Setup

### Clone Repository

git clone https://github.com/sairam633/markdown-notes

cd markdown-notes-app

---

## Backend Setup

cd backend

npm install

node server.js

Runs on:
http://localhost:5000

---

## Frontend Setup

cd frontend

npm install

npm run dev

Runs on:
http://localhost:5173

---

## API Endpoints

GET /notes

POST /notes

PUT /notes/:id

DELETE /notes/:id

GET /notes/search/:term
